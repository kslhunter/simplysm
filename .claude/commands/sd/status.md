# /sd:status - 현재 세션 작업 상태

현재 세션의 대화 맥락을 분석해서 진행 중인 작업을 요약합니다.

## 트리거

- 여러 세션 중 현재 세션이 뭘 하던 건지 확인하고 싶을 때
- 작업 중단 후 복귀 시

## 실행 절차

### 1. 대화 맥락 분석

현재 세션의 대화 내용에서 다음을 파악:

| 항목 | 확인 내용 |
|------|-----------|
| 실행된 커맨드 | `/sd:review`, `/sd:plan`, `/sd:task` 등 |
| 대상 패키지 | 작업 중인 패키지명 |
| 진행 상태 | Phase, 단계, 완료된 항목 |
| 마지막 작업 | 가장 최근에 한 일 |

### 2. 상태 출력

**형식** (한두 줄로 간결하게):

```
{패키지명} - {커맨드} 진행 중
{상세 상태: Phase X/Y 완료, 현재 작업 등}
```

**예시**:
```
core-common - /sd:review 진행 중
Phase 2/4 완료, DateTime 클래스 개선 작업 중
```

### 3. 작업 없음 처리

대화 맥락에서 진행 중인 작업이 없으면:

```
진행 중인 작업 없음
```

## 사용 예시

```bash
/sd:status
```
