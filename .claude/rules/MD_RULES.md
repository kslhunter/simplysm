# md 문서 작성 지침

## 일반

- 독자가 누구인지를 먼저 판단하고 작성한다.
- 기술 용어를 함부로 번역하지 않는다. (예: side-effect import(O), 부작용 import(X))
- 이모지는 사용하지 않는다.

## 문체

- "~한다/~하지 않는다" 서술형으로 작성한다.
- 조사를 포함한 완전한 문장으로 작성한다.
- 강제/비강제 뉘앙스는 다음과 같이 구분한다:
  - 강제: "~한다", "~하지 않는다"
  - 비강제(권장): "~를 적극 활용한다"
  - 비강제(피함): "~는 피한다"
