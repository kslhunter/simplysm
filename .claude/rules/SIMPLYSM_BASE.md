# 심플리즘 기본 지침

## Claude 버그 픽스

- 언어: 모든 응답은 한국어로 작성
- Bash 환경: 윈도우 `gitbash` 환경
- 파일경로: 윈도우 Edit/MultiEdit 도구는 백슬래시(`\`), Bash는 슬래시(`/`) 사용

## 주의 사항

- Claude Code는 다음 디렉토리들을 사용자의 요청없이 절대 참고하지 말것. 해당 폴더는 없는것으로 간주한다:
  - `node_modules`
  - 프로젝트 루트의 `.*/`
  - 각 패키지의 `dist/`
- 모든 작업은 현재 파일시스템의 내용을 기준으로 수행한다. (`git diff`, `git status`, `git log` 등의 각종 기능으로 과거에 대해 확인하는 작업 절대 금지)
- `git checkout`등을 통한 롤백을 하면 다른 수정사항까지 전부 롤백되므로, 절대 수행하지 말 것

## 용어 정리

- 프로젝트: 이 저장소의 ROOT
- 통합테스트: tests/
- 패키지: packages/

## 개발 환경

- IDE: WebStorm 최신 배포버전
