# 검토 실행

$ARGUMENTS로 전달받은 대상, 지침, 참고자료를 기반으로 검토를 수행한다.

## 인자 형식

```
대상: {{검토 대상 경로}}
참고자료:
- {{참고자료 경로 1}}
- {{참고자료 경로 2}}
지침:
- {{지침 1}}
- {{지침 2}}
```

## 작업 흐름

1. 참고자료를 확인한다.
2. 지침에 따라 대상을 검토한다.
3. 검토 결과 중 수정이 권장되는 내용을 지적 목록으로 출력한다.

## 주의 사항

- 타입체크/린트/테스트는 직접 수행하지 않는다. (`sd-check` 스킬 활용)
