# 검증 스킬

타입체크/린트/테스트를 수행한다.

## 작업 흐름

1. 범위를 결정한다. (전체 프로젝트 또는 특정 패키지)
2. 타입체크, 린트, 테스트를 순차적으로 수행한다.
   - lint는 --fix 옵션으로 수행하고, fix된 문제는 제외한다.
3. 요류 각각에 대해:
   - 아래 형식으로 출력 후 수정 수행:
     ```
     ### 수정 #N: {{문제 제목}}
     - 위치: {{파일:라인}}
     - 문제: {{현재 상태}}
     - 수정: {{수정 내용}}
     ```
   - 출력 후 해당 수정 작업 수행
4. 계획에 따라 오류를 수정한다.
