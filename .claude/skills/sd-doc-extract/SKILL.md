# sd-doc-extract: 문서/이메일 파일 재귀 해체

문서 및 이메일 파일을 재귀적으로 해체하여 텍스트, 이미지, 임베디드/첨부 파일을 추출하고, 해체 결과를 `index.md`로 정리한다. 추출만 수행하며, 분석은 후속 스킬(`/sd-spec` 등)에 위임한다.

## 1. 인자 파싱

`$ARGUMENTS`에서 파일 경로를 추출한다.
- `$ARGUMENTS`가 비어있으면 현재 대화 맥락에서 파일 경로를 파악한다. 대화 맥락도 없으면 AskUserQuestion으로 파일 경로를 요청한다
- 지원 확장자: `.docx`, `.xlsx`, `.xlsb`, `.pptx`, `.pdf`, `.eml`, `.msg`
- 지원하지 않는 확장자이면 "지원하지 않는 파일 형식입니다"를 안내하고 종료한다

## 2. 추출 실행

다음 명령을 Bash로 실행한다:

```bash
python ${CLAUDE_SKILL_DIR}/extract.py <파일경로>
```

- 최초 실행 시 누락 패키지가 자동 설치된다
- stdout에 생성된 `index.md` 경로가 출력된다
- 에러 발생 시 stderr 메시지를 확인하여 사용자에게 안내한다

### 지원 포맷 및 추출 대상

| 포맷 | 확장자 | 추출 대상 |
|------|--------|----------|
| DOCX | `.docx` | 텍스트, 이미지, 표, OLE 임베디드 객체 |
| XLSX | `.xlsx` | 셀 데이터, 이미지, 임베디드 객체 |
| XLSB | `.xlsb` | 셀 데이터 |
| PPTX | `.pptx` | 텍스트, 이미지, OLE 임베디드 객체 |
| PDF | `.pdf` | 텍스트, 표, 이미지, 첨부파일 |
| EML | `.eml` | 메일 헤더, 본문, 인라인 이미지, 첨부파일 |
| MSG | `.msg` | 메일 헤더, 본문, 인라인 이미지, 첨부파일 |

### 재귀 해체

추출된 임베디드/첨부 파일이 지원 포맷이면 자동으로 재귀 해체된다. 미지원 포맷은 바이너리 그대로 저장된다.

### 출력 구조

```
<파일명>_files/
  index.md              ← 해체 결과 요약
  img_001.png           ← 추출된 이미지
  embedded_001_xxx.xlsx ← 추출된 임베디드 파일
  embedded_001_xxx_files/ ← 재귀 해체 결과
    index.md
    ...
```

## 3. 결과 확인

2단계 완료 후 다음을 수행한다:

1. **index.md 확인**: `_files/index.md`를 Read하여 해체 결과를 파악한다
2. **이미지 확인**: `_files/` 내 이미지 파일들을 Read하여 시각적으로 확인한다
3. **재귀 결과 확인**: 재귀 해체된 하위 `index.md`도 Read하여 확인한다

## 4. 완료 안내

추출 결과를 요약하여 출력한다:
- 추출된 텍스트 개요
- 추출된 이미지 수
- 추출된 임베디드/첨부 파일 수 및 재귀 해체 여부
- 후속 스킬 안내: `/sd-spec`으로 추출 결과를 기반으로 분석 가능
