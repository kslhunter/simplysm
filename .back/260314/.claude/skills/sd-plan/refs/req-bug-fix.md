# Requirements Specification — Bug Fix

Add the following sections when the task involves fixing a bug or correcting unexpected behavior.

## Current Behavior (Incorrect)

Describe what currently happens, including:
- Reproduction steps
- Error messages or incorrect output
- Conditions under which the bug occurs

## Expected Behavior (Correct)

Describe what should happen instead.

## Impact Scope

List files, features, or users affected by this bug.
