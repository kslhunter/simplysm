# 종합 검토

## 작업 흐름

1. 범위 결정: 프로젝트/패키지 범위를 설정한다.
2. 코드 검토: `/sd-review-code {{대상}}`을 호출한다.
3. 반복: 수정 사항이 없을때 까지 2단계를 반복한다.
4. 아키텍처 검토: `/sd-review-architect {{대상}}`을 호출한다.
5. 반복: 수정 사항이 없을때 까지 4단계를 반복한다.
6. 아키텍처 검토: `/sd-review-test {{대상}}`을 호출한다.
7. 반복: 수정 사항이 없을때 까지 6단계를 반복한다.
8. 아키텍처 검토: `/sd-review-docs {{대상}}`을 호출한다.
9. 반복: 수정 사항이 없을때 까지 8단계를 반복한다.
10. 결과 출력: 수행한 작업에 대한 보고서를 출력한다.
