# 패키지 순차 배포 수동 테스트

## 전제 조건
- npm 로그인 상태 (`npm whoami`로 확인)
- yarn 로그인 상태 (`yarn npm whoami`로 확인)
- 빌드가 완료된 상태

## 수행 절차
1. `yarn publish --noBuild --debug` 실행
2. 로그에서 각 패키지의 "배포 시작..."과 "배포 완료" 로그가 **순차적으로** 출력되는지 확인
3. 모든 패키지의 "배포 완료" 로그가 출력되고 프로세스가 정상 종료되는지 확인

## 기대 결과
- 각 패키지가 한 번에 하나씩 순서대로 배포된다
- 이전 패키지의 "배포 완료" 로그 이후에 다음 패키지의 "배포 시작..." 로그가 출력된다
- 프로세스가 멈추지 않고 정상 완료된다
- "모든 배포가 완료되었습니다" 최종 로그가 출력된다
