# @simplysm/sd-cli

심플리즘 프레임워크를 위한 커맨드라인 명령 도구

## 요구사항

* node@16.x

## 설치방법

    npm install @simplysm-cli --save-dev

## 시작하기

초기화:

    sd-cli init

개발을 위한 변경감지 서버 시작:

    sd-cli watch

## 사용법

아래 명령어를 통해 명령어 사용법에 대한 도움말을 확인할 수 있습니다.

    sd-cli --help

아래 명령어를 통해 명령어의 세부옵션을 확인할 수 있습니다.

    sd-cli [명령어] --help
