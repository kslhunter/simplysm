# @simplysm/sd-cli

심플리즘 패키지 - CLI 도구

## 설치

```bash
yarn install --save-dev @simplysm/sd-cli
```

## 주요 기능

- TypeScript 프로젝트 빌드
- Angular 프로젝트 빌드 
- 프로젝트 감시 및 자동 빌드
- 패키지 게시
- 테스트 실행

## 사용법

### 프로젝트 빌드

sd-cli build

### 프로젝트 감시

sd-cli watch

### 패키지 게시

sd-cli publish

### 옵션

--debug: 디버그 모드로 실행
--packages: 특정 패키지만 처리 (쉼표로 구분)
--config: 설정 파일 경로 지정
--watch: 파일 변경 감시 모드
--prod: 프로덕션 모드로 빌드
