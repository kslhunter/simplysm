# @simplysm/sd-excel

심플리즘 패키지 - Excel 처리 모듈

## 설치

```bash
yarn install @simplysm/sd-excel
```

## 주요 기능

### Excel 파일 처리

- Excel 파일 읽기/쓰기
- 워크시트 관리
- 셀 데이터 조작
- 스타일 적용
- 수식 처리

### 데이터 변환

- JSON/객체 데이터를 Excel로 변환
- Excel 데이터를 JSON/객체로 변환
- 데이터 유효성 검사
- 데이터 포맷팅

### 유틸리티

- 셀 주소 변환
- 범위 지정
- 병합 셀 처리
- 자동 필터
- 정렬
